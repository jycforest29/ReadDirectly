# ReadDirectly
# 프로젝트 소개
인원: 1인 개발<br>
개발 기간: 4시간<br>
개요: 문장 직독직해 앱<br>
주요 기능 : 선택한 언어로 입력한 문장 직독직해 번역<br>
개발 동기 : 일본어 공부하려고 만들었음
# 사용한 기술
서버: DRF<br>
클라이언트: 안드로이드<br>
DB: sqlite3(장고 내장 DB)<br>
# 사용한 오픈소스
googletrans open api
